Translator API

  • Translates text (duh)
  • Can tell you if it can go from X to Y
  • Based on language packs
  • Available since Chrome 138
  • Requires User "Gesture"

Language Detector API

  • Detects language of input (duh)
  • Can tell you if a language can be detected
  • Returns a ranked list of languages with confidence (0 to 1)
  • Available since Chrome 138
  • Requires User "Gesture"

Summary API

  • Summarizes text (duh)
  • Multiple styles and length of summarization
  • Available since Chrome 138
  • Requires User "Gesture"

Summary Options

  • Types: key points (default), tldr, teaser, headline
  • Length: short, medium (default), long
  • Format: markdown (default) or plain-text
  • "sharedContext": additional info to help with summarization
  • Results sometimes includes context

Writer/Rewiter API

  • Generates or transforms text
  • Can specify a tone, length, format, and sharedContext
  • Can specify languages (input, output)
  • For length, in writer it is: short, medium, long. For rewriter, it is: shorter, as-is, longer
  • For writer and tone: formal, neutral, casual. For rewriter and tone: more-formal, as-is, more-casual
  • Origin Trial (flag needed)

Proofreader API

  • Identifies grammar, spelling, and punctuation errors
  • Is this better than what's in the browser/OS? 🤷‍♀️
  • Can do correction, add labels, and explain issues*
  • Can specify input language
  • Can NOT modify correction types/explanations
  • Origin Trial (flag needed)

The Prompt API

  • General purpose prompting API
  • Incredibly flexible (system instructions, structured output, session persistence/cloning)
  • Session-based API
  • Can work with images and audio
  • Origin Trial (released for extensions)

	let translatorCapabilities = await window.Translator.availability({
		sourceLanguage: 'en',
		targetLanguage: 'zh'
	});

	console.log('English to Mandarin? ', translatorCapabilities);
}
